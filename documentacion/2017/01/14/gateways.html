Gateways

Los Gateways o Puertas de Enlace, son la conexión primaria entre cada una de las empresas asociadas que se encuentren en diferentes localidades. Por lo tanto los cambios que se realicen en estos deberán ser acordes a cada sucursal.

Las funciones principales son:

  • Regular el Internet para que sea utilizado solo en forma productiva para la empresa
  • Realizar una conexión virtual privada (VPN) entre sucursales
  • Monitoreo de las conexiones entrantes y salientes de cada sucursal

Desbloquear una URL:

Esto no aplica a SEBO01 ya que ellos no tienen ningún tipo de bloqueo

  1. Ingresar en el Gateway al cual se le quiere desbloquear la URL

  2. Seguir la ruta:

    1. Gateway -> Content Filter and Proxy -> Content Filter Engine
    2. Configure Policy
    3. Exception Sites -> Edit
    4. Add
    5. Agregar la URL base que se desee desbloquear

Nota:

  • Solo es necesario agregar la URL base (el texto directamente a la izquierda de “.com”) Ejemplo: Si la URL es http://www.compresoresservicios.com/blog/ la base de esta sera compresoresservicios.com

  • Si desea verificar que una URL este desbloqueada debe buscarla en el paso 2.C:

  • Si desea eliminar que una URL este desbloqueada debe buscarla en el paso 2.C y luego presionar “Delete”.

Observar el Ancho de banda usado Actualmente:

Esta opción permite ver el ancho de banda usado por la respectiva sucursal.

  1. Ingresar en el Gateway en el cual se desee observar el Ancho de Banda

  2. Seguir la ruta: 2.A Reports -> Performance and Resources -> Bandwidth Viewer

  3. Dependiendo del Gateway realizar: 3.A CSBO01 Dejar solamente las casillas “enp1s0” estas representan la conexión con CANTV

Notas:

  • Sent: Representa el ancho de banda de subida (Upload)
  • Recv: Representa el ancho de banda de descarga (Download)

Ejemplos:

  • Si el consumo es alto, la velocidad de conexión podría verse afectada.
  • Si el consumo es bajo pero la velocidad de conexiona es lenta el problema puede ser directamente la conexión con el proveedor de Internet (CANTV, Digitel…).

Determinar la computadora que esta consumiendo mayor ancho de banda:

  1. Ingresar en el Gateway en el cual se desee observar el Ancho de Banda
  2. Seguir la ruta:
    1. Reports-> User Managment -> Filter and Proxy Report

    1. Top IPs

    Obserar la IP que tenga mayor consumo

  3. Comparar la IP obtenida con la hoja de calculo “Mapa de Red.xlsx”, en la respectiva pagina
